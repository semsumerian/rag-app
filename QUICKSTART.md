# Быстрый запуск RAG Application

Краткая инструкция для повседневного использования.

## ⚡ Быстрый старт (3 терминала)

Перед запуском проверьте, что команды доступны:

```bash
node -v
npm -v
```

### Терминал 1 - Сервер:
```bash
cd server
npm run dev
```

### Терминал 2 - Клиент:
```bash
cd client
npm run dev
```

### Терминал 3 - Туннель (для доступа извне):
```bash
cloudflared tunnel --url http://localhost:5173
```

**Готово!** Открывайте URL из вывода туннеля.

---

## 📋 Чек-лист перед запуском

- [ ] Node.js LTS установлен
- [ ] LM Studio запущен с моделями
- [ ] Сервер API в LM Studio активен на порту 1235
- [ ] (Опционально) cloudflared установлен

---

## 🔧 Если что-то не работает

**Ошибка "npm не найден"**
1. Проверьте Node.js: `node -v`
2. Если команда не найдена, установите Node.js LTS
3. Перезапустите терминал после установки

**Ошибка 502 Bad Gateway**
1. Проверьте, что клиент запущен
2. Перезапустите туннель

**Туннель не создает URL**
```bash
# Проверьте, что cloudflared установлен:
cloudflared --version
```

---

## 🌐 Доступ к приложению

- **Локально**: http://localhost:5173
- **Извне**: URL из вывода туннеля (https://xxx.trycloudflare.com)

---

## 🛑 Остановка

Остановите процессы через `Ctrl+C` в каждом терминале.

---

## 💡 Советы

- Оставляйте терминалы открытыми - приложение работает пока процессы запущены
- URL туннеля меняется каждый раз - используйте новый
- Данные (документы, чанки) сохраняются между запусками

---

## 🪟 Команда туннеля для Windows

```powershell
cloudflared.exe tunnel --url http://localhost:5173
```
